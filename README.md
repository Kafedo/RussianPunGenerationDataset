# Разработка нейронной сети и набора данных для генерации каламбура
*ВКР Федорищевой Екатерины*

**В данном репозитории содержатся:**
1) Собранный и размеченный датасет из 931 каламбуров на русском языке, основанных на омонимии и полисемии (*Corpora/*)
  * *full_annotated_corpus.json* -- основной блок размеченных данных
  * *full_maybe_corpus.json* и *full_worst_corpus.json* содержат каламбуры, отсеянные по качественным или цензурным соображениям
2) Результаты генераций, полученных в процессе оценки моделей, в 4 блока: исходная модель и дообученная, свободная генерация и по заготовленным многозначным глаголам (*Gen_puns/*). 
3) Дообученная модель выложена на [HuggungFace](https://huggingface.co/kafedor/Russian_pun_generation_qwen3.5_adapter).
